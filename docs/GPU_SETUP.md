# GPU Setup

Venture Trial T0 was executed on an NVIDIA GeForce RTX 5070 with compute
capability `sm_120`, PyTorch `2.13.0+cu132` and an NVIDIA driver exposing CUDA
13.3.

Install the CUDA-enabled PyTorch wheel after the base project dependencies:

```powershell
python -m pip install --force-reinstall --no-deps `
  torch==2.13.0+cu132 `
  --index-url https://download.pytorch.org/whl/cu132
```

Verify the exact device path before training:

```powershell
python -c "import torch; print(torch.__version__); print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0)); print(torch.cuda.get_device_capability(0)); print(torch.cuda.get_arch_list())"
```

The registered trial configuration uses `device: auto`; it selects CUDA only
when `torch.cuda.is_available()` is true. CPU-only CI remains supported, so the
CUDA wheel is intentionally not hard-coded in `pyproject.toml`.

Use the current command generated by the official
[PyTorch installation selector](https://docs.pytorch.org/get-started/locally/)
when reproducing on a different driver or GPU generation.
